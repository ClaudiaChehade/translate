OTA2 Online Calibration Test

Test Case Description

  • If the following description is too subjective, the confidence of the 2D lane detection can be used as an objective standard

Road conditions

good

  • High -speed, elevated, urban fast roads (flat roads)

  • The three lanes live in the middle, and the lane is clear

  • Non -congestion period

bad

  • General urban roads (bumpy roads)

  • Below the three lanes (parking lots), the lane line is unclear

  • Morning and evening peaks, congestion periods (there are many surrounding vehicles, lane lines are blocked)

Weather conditions

good

  • Sunny and cloudy days

bad

  • At night, rainy days (small/medium/large), heavy fog, backlight

Driving habit

good

  • The speed 40 ~ 100 km/h

  • Driving smoothly, try to go straight to avoid changing lanes

bad

  • Too low or too high

  • Frequent changes, rapid acceleration, rapid deceleration, sudden braking, rapid turning

  • The Test Case listed below is not fully covered. If the test time allows it to be supplemented (Test at night 1, 5 and 7 at night)

  • The inter -group test is independent and cannot be carried out within the same power -on cycle

  • Each group of test duration requires a single camera convergence of more than 3 times (Full Good is expected to be 30 minutes, when the bad or yaw disturbance is large, convergence will take time to increase)

  • Based on the Office Calib disturbing YAW, you must remove the online calib parameter in Persistency before each test

  • Turn off Wenyuan's online correction module before testing

1. How to disturb offline calib parameter

disturb_calib_param.py

  • python disturb_calib_param.py --folder_path=[xxtriniic_folder] --sensor_name=[sensor_name] --delta_yaw=0.0 --delta_pitch=0.0 --delta_roll=0.0

  • When the t_v_c in the calibration parameter rotates in order of Euler Corner XYZ, the pitch angle under the coordinate system of the vehicle corresponds to ROLL, so the disturbance of Pitch needs to be applied to the delta_roll;

  • The script is the calibration file in the folder. It is best to copy a calibration parameter folder to run the script every time you disturb;
