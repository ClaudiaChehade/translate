xPER safety build plan

  • Plan from model perspective

From model perspective, we try to migrate model by model to the safety build version.

For the first model, we will verify the custom operators we developed, Inference model in orin and get KPI, Check the KPI, if KPI is good, then we will continue to migrate next model. If the KPI is terrible, we will feed back to model team, ask them to adapt their model structure according to dos6.0.7 supported layers, retrain their model.
