Universal data distribution requirements

We will establish a new data set to support large -scale mass -produced L2 ++ light map autonomous driving products.The product will be used under different time, weather, road conditions, lighting conditions and sensor working conditions across the country.Therefore, we need to ensure that the establishment of a data set has been carefully designed and monitored to cover different situations.We have two levels of labels.One is a general label, which is important for all different tasks. For example, weather/sensor conditions should affect all tasks.The other is a specific task label. For example, for lane line detection tasks, it will focus on the narrow and long shape of the lane, as well as covering wear, etc. This situation is more like creating a specific data set for each Corner Case.
