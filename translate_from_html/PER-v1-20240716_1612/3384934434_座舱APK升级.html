The cockpit lion navigation map upgrade (ensure the start of the USB debugging mode)

  • 1. Uninstallation of the original software (ensure the operation, two uninstall)

    • 1.1

      • adb root

      • adb remount

      • adb shell uninstall com.lion.appfwk.navi

    • 1.2

      • adb root

      • adb remount

      • adb shell

      • cd /vendor/app/RedwoodLionNavi/

      • rm *.apk

  • 2. Delete the original data

    • adb root

    • adb remount

    • adb shell

    • cd storage/emulated/0/Android/data

    • rm -rf com.lion.appfwk.navi

  • 3. Installation

    • adb root

    • adb remount

    • adb shell push *.apk /vendor/app/RedwoodLionNavi/

    • adb shell reboot
