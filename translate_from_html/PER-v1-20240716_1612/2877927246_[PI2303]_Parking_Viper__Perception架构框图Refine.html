[PI2303] Parking Viper :: Perception architecture box diagram Refine


VIPER :: Perception architecture optimization mainly considers the following points:

  • The CPU calculated dense task is set to an independent Activity and deployed on an independent CPU Core

  • CPU calculates the mission of light load and try to merge into an Activity, maximize the use of the current CPU core resources

  • Combined with performance considerations, the Image Data obtained from the NVMedia 2D Engine will be processed by the GPU.

  • In order to decompose FEV and BEV (IPM) processing links, FEV and BEV (IPM) are designed as two independent Activity
