HDMapService Event Code

Background synchronization

Each algorithm module will be abnormal. For these abnormalities, we need to consider the design monitoring mechanism in the future.Report an error status code on the system interface.

Based on this background, the current SYS group requires each algorithm module to sort out the various abnormalities inside the algorithm. This job must be iterative improvement.The PI2303 system requires us to output a version of information, and then the demand will be established in Door. After the demand is confirmed by the Review, the algorithm logic monitoring function will be performed in the subsequent PI.
