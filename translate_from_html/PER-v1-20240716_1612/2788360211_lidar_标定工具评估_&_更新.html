Solution: Use "two frame data" calibration

Frame 1 data: Collect Front-Tele-Camera image at the door of the calibration, Main-Lidar and Front-Lidar

  • Use the installation design value to convert the point cloud from the lidar coordinate system to the vehicle coordinate system

  • Use the rectangular ROI to filter out the ground point cloud

  • Perform RANSAC plane fitting, solve the method vector

  • Pitch and roll through the method of the method, and calculate Z, as the initial value and threshold

Frame 2 Data: Collect the vehicle sensor data normally on the middle institution

  • Based on the first frame data, fix Pitch, ROLL, and Z, optimize YAW, X, and Y

  • If the Pitch, ROLL, and Z in the calibration result are large from the data of the first frame, it is determined to be the definition failure
