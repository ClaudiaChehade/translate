3.1.0-4.0-1 Step4.1: The system shall execute HPA training function after driver select training.

1) System shall check the vehicle status and remind the corresponding info, such as slow down, direction of moving, record remaining distance, typical object display, etc.

2) Start training remind and vehicle start to driving by driver, the system record the corresponding path, feature map, end point etc.

3) Driver press the button to finish training process and the system shall define the training path with Line Parking according to end position of the vehicle in the parking space marked with line, otherwise, the space position will be defined. (FOR Line PS, the system shall support to find the near by parking space marked line for parking in in case of specific parking space blocked, FOR space position, the system shall support to search nearby space position in case of specific position blocked by others object (Figure shown)

4) Optimize training path then store in NVM/EMMC(Training finish). Then display the whole path including start/stop point on UI and remind driver to name the path.

Notice:: during HPA training stage, the APA/RPA could be supported at the same time.

  • Vector information (incl. PS, arrow, lane, etc.)

  • Dymanic object (moving status)
