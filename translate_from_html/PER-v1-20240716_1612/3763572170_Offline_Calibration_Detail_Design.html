lidar calibration algorithm

Input

Lidar Calib Data (add Gaussian noise)
GT cloud poi nt (High-precision scanner)
circle center 3D groundtruth

Output

lidar extrinsic

Algorithm

  1. According to the design value of the LIDAR installation location, the point cloud is converted to the coordinate system of approximate car body, and ROI filtering is performed to retain the point cloud near the designated board area

  2. Using ICP, the standard point cloud and the true value of the point cloud are in the initial standard, and the initial exogenous parameters are calculated

  3. After the clustering class is divided into multiple planes, use RANSCA to fit plane and calculate the method of a plane

  4. Treatment of each plane, extract the edge of the round hole, combine the plane method vector to calculate the center of the circle
    According to the vertical resolution of the lidar, the line beam is divided by the point cloud, and then the point clouds in each beam are sorted in the bias of the horizontal direction. The distance of the adjacent point cloud is calculated.When setting the threshold, judge the point cloud at the edge of the round hole


    In contrast to the 3D point of the real value, a classification of the extracted edge point cloud is classified


  5. Select 1 point as the benchmark, check the distance from several other rounds to ensure that the relative position is in line with the actual scene


  6. 3D-> 3D point cloud standard, minimize errors, optimize lidar external parameters
