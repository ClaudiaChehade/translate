The following content takes NPOS as an example:


1 Combination navigation device calibration, and coordinate system settings include two parts: Rotation from Imu Body to Vehicle (RBV) and Translation


2 RBV

(1) What is the body coordinate system defined by the combination navigation equipment, and the nPOS body coordinate system is right-front-upper.

(2) Execution INSCALIBRATE RBV NEW ï¼Œ If the execution of the relevant manual is successfully executed, the INS output result (INSPVAX, etc.) is the nPOS-defined car body coordinate system right-front-upper.



3 translation

(1) Set command: SetInstranslation User TX TY TZ 0 0 0 0 // Translation from the Imu Center of Navigation to the User Output Location.

(2) If the TX TY TZ set in the above command is 0, then the INS output result (INSPVAX, etc.) is the position of the original point of the NPOS coordinate shaft;

(3) If the INS output result (INSPVAX, etc.) indicates the position of the center center center of the vehicle, you need to configure the Translation of the NPOS and rear wheels in the center of the rear wheels.
