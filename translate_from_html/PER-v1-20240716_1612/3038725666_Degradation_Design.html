Degradation Design

1 Degradation Conception

Refer to SYS's definition of downgrade strategies: Degradation Concept for Wave3

For the positioning module, Fan mainly focuses on F Ailure Cluster, which needs to pay attention to the Failure Detection for internal development

2 Degradation Data Flow

  1. Step: Asw generates fault event sent to ARA :: Com Gateway to generate FID, and FCT receives all FIDs to downgrade the system.

  2. Upstream: ASW receives interested FID to help generate Fault Event.

3 Xloc Degradation Architecture

Because there are many internal modules of LOC and interactive interactions, SOC and MCU are distributed, so the Fault List inside the LOC needs to be combed.

  1. LOC Fault Event requires all relegation time to produce abnormal functions.

  2. The internal Fault List of LOC can cover all detected abnormalities, which can help the LOC submucting downstream processing. Passing based on algorithm communication interfaces, it is more effective.
